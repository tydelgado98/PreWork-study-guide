HTML

  • The head elements contains information about the website
  • The body element represents the visible content shown to the user
  • The HTML gives the content structure

CSS

  • The CSS gives the content style
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • "*" Applies a rule to the all the elements visible on the page

Git

  • A system that lets you manage and keep track of each version of your project
  • Git status: checks what branch we are currently on
  • Git checkout -b "branch-name": creates a new branch and switches to it
  • Git commit -m "add starter code": saves branch changes

JavaScript

  • The JS allows users to interact with a website
